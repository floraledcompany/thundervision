FAQ

What is this app for?
This app is meant to be a quick and easy way to create consistent project directories for all new projects. Simply type in the project title, select a destination folder (i.e. '2019') either by typing in a path or using the "Browse" button, and click create!

What is the template location and project type options for?
If desired, a project type can be selected. When the project is created, template project files named after the project will be placed in the appropriate folders within the new project directory (thats a lot of 'project's!). You can choose general options like "After Effects", or you can populate your project file with more specific parameters by choosing "South Board" or "Ribbons", for example.
If a template project is selected, a valid path to the template folder must be given in the Template Location text field. This can be selected using the "Browse" button or by manually typing a path into the text field.
PLEASE NOTE: If a project type is selected, a valid template path MUST be given in order to click 'Create Project'.

What do the colors in the text fields mean?
A green field indicates it is valid. White means required but not yet valid. Gray means not currently required (for example, if no project types are selected, the template location is not required). When a text field is selected, orange indicates invalid. As soon as the field is valid, it will switch to green.
